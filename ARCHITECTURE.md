## ğŸ”Œ API Endpoints

### `POST /api/upload`
Upload video and generate story.

**Request:**
```
Content-Type: multipart/form-data

Fields:
- video: File (required) - Video file (mp4, mov, avi, mkv, webm)
- personas: String (optional) - Character descriptions for story context
```

**Response:**
```json
{
  "success": true,
  "message": "Video processed and story generated!",
  "jobId": "uuid-here",
  "summary": {
    "scenesDetected": 34,
    "audioSegments": 34,
    "keyframesExtracted": 102,
    "scenesAnalyzed": 34
  },
  "story": "Generated story in Bahasa Melayu..."
}
```

### `GET /api/job/:jobId`
Get job processing result.

**Response:** Contents of `result.json`

### `GET /api/job/:jobId/story`
Get generated story for a job.

**Response:**
```json
{
  "story": "Generated story text..."
}
```
